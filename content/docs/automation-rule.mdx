An Automation Rule is a Growth Center V1 rule that supports opportunity-triggered marketing workflows. It can suggest or create drafts, but it does not auto-publish.

## Define Automation Rule in Growth Center

Growth Center Automations V1 includes rules and runs. Publisher roles can create automation rules, preview behavior, and observe automation runs.

Automation rules belong to the `/marketing` product area, not the legacy CRM `/automations` area.

## Keep Automation Rules Draft-Centered

Automation Rules can support suggest-only behavior or auto-create-draft behavior.

When an Automation Rule creates a draft, that draft still needs normal review and explicit publishing.

## Exclude Auto-Publish From V1 Rules

Growth Center Automations V1 does not auto-publish. Selective auto-publish after review maturity is future scope, not current V1 behavior.

Scheduled or scanner automation triggers beyond opportunity-facing V1 are also deferred scope.
