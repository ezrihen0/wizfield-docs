WizField Docs is the public knowledge layer for WizField product users, buyers, and operators. It explains how WizField works, how users should navigate product workflows, and where important product boundaries exist.

## Use WizField Docs to Understand Product Behavior

WizField Docs will publish structured pages for onboarding, product foundations, AI behavior, Growth Center, Language Store, billing, troubleshooting, and glossary terms.

## Trust Published WizField Product Truth

Every public docs page must be based on verified product behavior or canonical internal Source of Truth documents. WizField Docs must not publish raw internal planning files, engineering verification detail, or roadmap work as current product capability.

## Follow the Docs Foundation Roadmap

The first public content set will focus on the foundation pages approved in the internal content backlog. Full category expansion will happen only when each category has real content ready to publish.
