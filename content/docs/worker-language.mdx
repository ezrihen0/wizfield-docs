---
title: Worker Language
description: Understand a Worker Language as an organization-enabled internal language preference for WizField staff, separate from customer-facing English output.
---

A Worker Language is an internal language option enabled for an organization. Users can choose a preferred worker language within the active organization.

## Define Worker Language in Language Store

Language Store has a workspace language layer where the organization decides which worker languages are enabled.

Each user can select a personal preferred language from the languages enabled for the active organization.

## Keep Worker Language Organization-Scoped

The same user may use different preferred languages in different organizations. Worker language behavior follows the active organization, not one global user setting.

Language activation and usage enforcement are organization-scoped.

## Fall Back to English When Needed

English is always available and does not consume a paid language slot.

If an organization disables a language that a user previously selected, WizField must safely fall back to English for that organization.
